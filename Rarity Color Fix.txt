#<Rarity Color Fix>

    #NoMoreJebaited

    By Koby

    Image Album: https://imgur.com/a/AEZNy

    Changes the rarity color of unique aftermarket items to a dark teal to be more distinguishable from other items.

    Changes the color of Moxxi weapons to a blood red.

    Supports Gemstone weapons from the Unofficial Community Patch.

    Changes the order of items in the inventory: Pearl -> Seraph -> (Unique Uniques if enabled) -> Legendary -> Moxxi -> Unique -> Gemstone -> E-Tech -> Very Rare...

    Has an optional module to turn all Heads and Skins to a grey-ish color.

    Has an optional module to turn all Legendary items which don't world drop into Dark Orange items (Unique Uniques).

    Also changes the color of Moxxi weapons to a blood red.

    Has 2 money color options: Mint green money (default) and normal yellow.

    Full Filter Tool support!

    Thanks to Dave for providing the Rarity Colors resource.

    Also thanks to FromDarkHell for writing me a script for the Heads and Skins.

    INCOMPATIBILITY NOTE:

    This mod will most likely conflict with any mod which changes rarity colors and rarities of items.

    If you have basic modding and reading knowledge, you can fix the rarity in the weapon/item mod yourself.
